# Screen Rules

## Login screen

- no bottom navigation
- centered layout
- logo / branding area
- title and subtitle
- primary login action
- secondary login action
- lightweight sign-up link

## Chat list screen

- bottom navigation visible
- search / list browsing
- recent conversations list
- category chips if needed
- CTA for starting a new chat

## Create senior screen

- bottom navigation visible
- form-like composition
- structured sections
- selection controls for gender / age / occupation / income
- primary CTA near bottom

## Chat screen

- no bottom navigation
- app bar visible
- message list + input bar layout
- - action opens popup menu
- popup options:
  - 先輩を追加する
  - 自己分析を更新

## Dashboard screen

- bottom navigation visible
- summary metrics at top
- categorized insight sections
- scrollable layout
- reusable cards and chips

## Settings screen

- bottom navigation visible
- simple grouped settings layout
- user/account actions accessible

## Self-analysis update feedback

When user triggers 自己分析を更新:

- show a lightweight toast/snackbar
- do not interrupt conversation flow
- do not open a large modal unless explicitly required
