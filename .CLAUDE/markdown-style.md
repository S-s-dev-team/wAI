# Markdown スタイルガイド (markdownlint 準拠)

このプロジェクトの markdown ファイルは markdownlint に準拠すること。

## ルール一覧

- **MD040**: コードブロックには必ず言語を指定する (例: `bash`, `go`, `text`)
- **MD060**: テーブルのパイプ `|` の周辺に適切なスペースを入れ、列幅を揃える (style: "aligned")
  - 日本語の全角文字は表示幅2カラム分を占めるため、半角スペースでのパディング量を調整して末尾パイプの表示位置をヘッダーと揃えること
  - 全角1文字 = 半角2文字分の幅として計算する
- **MD022**: 見出しの前後に空行を1行ずつ入れる
- **MD031**: コードブロックの前後に空行を入れる
- **MD032**: リストの前後に空行を入れる
- **MD034**: bare URL は使わず、マークダウンリンク形式 `[text](url)` または `<url>` で記載する
