# wAI

> 就活という「新しい世界」への一歩を、AI先輩たちと一緒に。

---

## 概要

**wAI** は、就活を始めたばかりの学生が **AI先輩との会話** を通じて、自己分析・業界研究・企業研究をスムーズに進められるプロダクトです。

ハッカソンテーマ **「Brand New "Hello World"」** をもとに、未来の選択を迫られる「就活」という新しい世界に踏み出す学生たちをサポートするために生まれました。

## 解決したい課題

- 就活を始めたばかりの学生は **何から手をつければいいか分からない**
- 自己分析や業界研究は孤独で、**相談できる先輩が身近にいない**
- キャリアセンターや就活サイトの情報は画一的で、**自分ごとにしにくい**

---

## 強み・特徴

### 🧑‍🎓 先輩AIとのチャットで自己分析

チャット開始時に設定した「先輩AIペルソナ」と1対1で会話しながら、自己分析・業界研究・企業研究を自然に進められます。

### 👥 グループチャット形式でペルソナ追加

会話の途中で別のペルソナ（プリセットの先輩）を呼び出せます。複数の視点からアドバイスを得られる、グループチャットのような体験です。

### 💡 気づきがダッシュボードに蓄積

AIとの会話を通じて浮かび上がった「気づき」が自動でダッシュボードに記録されます。自分の思考の変化をいつでも振り返ることができます。

## セットアップ

### 必要な環境

- [Docker / Docker Compose](https://www.docker.com/)
- [aqua](https://aquaproj.github.io/)（CLI ツール管理）
- [Flutter SDK](https://flutter.dev/)
- Go 1.21+

### 起動方法

```bash
# aqua で CLI ツールをインストール
aqua install

# Docker 環境を起動（バックエンド + Swagger UI）
make up

# Swagger UI を確認
open http://localhost:8081
```

### Makefile コマンド一覧

| コマンド         | 説明                                       |
| ---------------- | ------------------------------------------ |
| `make up`        | Docker 環境を起動                          |
| `make down`      | Docker 環境を停止                          |
| `make ps`        | ステータスを確認                           |
| `make gen`       | OpenAPI からコードを再生成（Go + Flutter） |
| `make gen-back`  | Go のサーバースタブを生成                  |
| `make gen-front` | TypeScript の型定義を生成                  |
| `make docs-up`   | Swagger UI を起動                          |
| `make server`    | サーバーをローカルで起動                   |
